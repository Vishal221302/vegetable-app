A React Native application for browsing products, managing cart, and secure checkout with a multi-address management system.

## 📱 Features
- **Premium UI**: Smooth animations and card-based design.
- **Address Management**: Save multiple Home/Office addresses with GPS support.
- **Real-time Cart**: Persistent cart synchronized with the backend.
- **Order Tracking**: View order history and current status.

## 🛠️ Setup
1. `npm install`
2. Update `src/utils/api.js` with your backend IP address.
3. `npx expo start`

## 📦 Tech Stack
- React Native (Expo)
- Redux Toolkit (State Management)
- React Navigation
- Axios
- Lucide/Ionicons Icons
